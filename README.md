# Rock, Paper, Scissors

A simple Rock, Paper, Scissors game built with Python and Tkinter.

## Screenshot

![Screenshot](Screenshot%202026-06-15%20044336.png)

## Requirements

- Python 3.x
- Tkinter (included in standard Python installations)

## Usage

1. Select a move from the dropdown.
2. Click **Confirm**.
3. The computer's move and result are displayed instantly.

## How It Works

- The player selects Rock, Paper, or Scissors from a dropdown menu.
- The computer picks a move at random.
- Win/Loss/Draw is determined and color-coded green, red, or grey.
